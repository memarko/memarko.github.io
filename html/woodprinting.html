Wood Printing

Printing something on wood can be done by using a modified ordinary inkjet printer.

There are a few drawbacks. The thickness of wood must be very precise. Because greater distance from cartage can make picture to get out of focus. If cartage is too close to the wood it can become damaged because it is scratching on the wood. Picture become out of focus because ink is ejected on different angle from the cartage.

I used my old printer HP 710C. It is so old that it has LPT port only. So I bought USB to LPT adapter. Linux drivers are unfortunately not working. Fortunately it works on Windows without any additional drivers. When I tried to print picture in Linux it did not put any ink on the wood. Despite it did move the head left and right and also push wood inside as it was printing something. So it took me a while I figure it out what was wrong.
